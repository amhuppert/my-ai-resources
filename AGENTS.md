# Project Standards

## Memory Bank

- Read all memory bank files at task start
- Update `memory-bank/focus.md` when completing tasks or changing direction

## Documentation

- Keep AI agent instructions concise and machine-actionable

## Code Standards

- Avoid over-engineering; make only requested changes

## Commands

- `ai install --scope user` - Install binary utilities, agent-docs, MCP servers
- `ai install --scope project` - Install plugin, Cursor rules, project configs
- `cd typescript && bun run build` - Compile TypeScript tools
